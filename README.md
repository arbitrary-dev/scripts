### cnv

Utility for convenient media convertion.

Requires these gentoo packages:
- media-gfx/imagemagick
- media-video/ffmpeg x264 vpx fdk
- sys-process/parallel

And license:
- media-libs/fdk-aac FraunhoferFDK

### dominant-color
