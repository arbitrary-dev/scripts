README
======

### bkrn.sh

Helps to build kernel.

	// Self-descripting ones.
	bkrn.sh prepare
	bkrn.sh build
	bkrn.sh initramfs
	bkrn.sh install
	bkrn.sh backup
	bkrn.sh restore

### cnv.sh

Utility for convenient media convertion.

	// Type these to get usage descriptions.
	cnv.sh ao
	cnv.sh photos
	cnv.sh videos
	cnv.sh player
	cnv.sh webm
