linux-scripts
======

### bkrn

Helps to build kernel.

  // Self-descripting ones.
  bkrn prepare
  bkrn build
  bkrn initramfs
  bkrn install
  bkrn backup
  bkrn restore

### cnv

Utility for convenient media convertion.

  // Type these to get usage descriptions.
  cnv ao
  cnv photos
  cnv videos
  cnv player
  cnv film
  cnv webm

### dominant-color

Returns dominant color of supplied image.

  dominant-color image.jpg
